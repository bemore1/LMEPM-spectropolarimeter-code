This repository contains the MATLAB implementation for the numerical methods presented in our paper: "Ultrahigh-resolution computational spectropolarimeter by harnessing a leakage-mode-encoded photonic molecule". The core of this toolkit is a solver for nonlinear systems of equations leveraging Compressed Sensing (CS) principles via the CVX optimization framework. It is designed to handle underdetermined nonlinear problems in computational system.

Prerequisites MATLAB (Tested on R2022b) CVX Bundle (Installed and configured in the MATLAB path)
